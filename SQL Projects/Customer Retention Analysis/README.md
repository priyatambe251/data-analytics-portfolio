# Customer Retention Analysis

## 📌 Objective

Analyze customer purchasing behavior to identify loyal customers, repeat buyers, and opportunities to improve customer retention.

---

## 📂 Dataset

### Tables Used

- Customers
- Orders

---

## ❓ Business Questions Solved

1. Find repeat customers.
2. Calculate total spending by each customer.
3. Identify one-time buyers.
4. Rank customers by spending.
5. Classify customers into loyalty segments.
6. Analyze monthly customer activity.
