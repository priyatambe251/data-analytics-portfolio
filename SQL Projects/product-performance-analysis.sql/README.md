# Product Performance & Inventory Analysis

## 📌 Objective

Evaluate product sales performance and inventory trends to identify best-selling products, slow-moving items, and category performance.

---

## 📂 Dataset

### Tables Used

- Products
- Orders

---

## ❓ Business Questions Solved

1. Find total quantity sold by each product.
2. Identify top-selling products.
3. Find products that were never ordered.
4. Analyze category-wise sales.
5. Rank products by revenue.
6. Generate monthly sales reports.
